### 📗使用方法

#### 🔑常规部署

 - 安装Python3.6+环境
 - 下载并解压项目代码包
 - 修改`config.yml`文件中的相关配置内容
 - 运行`pip install -r requirements.txt -t ./ -i https://mirrors.aliyun.com/pypi/simple`安装项目依赖
 - 执行`Python index.py`即可运行项目
